# Day 1 – JavaScript Basics: Variables & Data Types

Today I revised the fundamentals of JavaScript variables and data types.  
The focus was on understanding how to store and manipulate different types of values such as numbers and strings.

## Concepts Practiced
- Declaring variables using `let` and `const`
- Working with number and string data types
- Performing basic arithmetic operations
- Logging results using `console.log()`

## Mini Projects Built

### 1. Temperature Converter
Converts temperature from Celsius to Fahrenheit using JavaScript variables and basic arithmetic.

### 2. Simple Calculator
Performs basic math operations such as addition, subtraction, multiplication, and division.

### 3. Age Calculator
Calculates a person's age based on their birth year and the current year.

## What I Learned
- How variables store values
- The difference between `let` and `const`
- How JavaScript handles numbers and strings
- Using variables to calculate and display results
